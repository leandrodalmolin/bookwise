## Requirements

- Node v20.1.0
- pnpm v8.15.5

## Project Setup

Follow these steps to set up the project:

```bash
pnpm install
```

### Start dev server

```bash
pnpm run dev
```

### Build for production

```bash
pnpm run build
```

### Preview production build locally

```bash
pnpm run preview
```

### Lint files

```bash
pnpm run lint
```

### Format all files with Prettier

```bash
pnpm run format
```
